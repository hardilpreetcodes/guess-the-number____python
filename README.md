# guess-the-number____python
A Python number guessing game built using core python concepts to have some fun!
Author- Hardilpreet Kaur

## How the Game Works
- The program generates a random number between 1 and 100
- The user guesses the number
- The program gives hints: too high or too low
- The game ends when the correct number is guessed

## Concepts Used
- Variables
- Loops
- Conditional statements
- Random module
- User input

## How to Run
```bash
python number_guessing_game.py
